# Project Reporting System (PRS)

A centralized platform for managing projects across multiple departments with full lifecycle tracking and role-based security.

## Overview

PRS streamlines project management from start to finish, guiding projects through initiation, approval, execution, and delivery across Administration, Studio, Workshop, and Warehouse departments. The system ensures transparency and accountability with secure role-based access control.

## Tech Stack

**Frontend:** React.js, Tailwind CSS, Boxicons  
**Backend:** Node.js, Express.js, MySQL  
**Security:** BCRYPT encryption, TLS protection

## Features

- **Project Lifecycle Management** - End-to-end tracking from initiation to delivery
- **Role-Based Access Control** - Secure permissions for different departments and users
- **Automated Email Notifications** - Real-time alerts for project updates
- **Multi-Department Workflow** - Seamless collaboration across teams
- **Real-time Status Tracking** - Live updates on project progress

## Security

- Password hashing with BCRYPT
- TLS encryption for data protection
- Role-based permissions (RBAC)
- Secure REST API architecture
